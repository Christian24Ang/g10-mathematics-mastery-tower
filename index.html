Rules

How to Play

Perfect Run

Answer correctly to move up one sublevel. One wrong answer ends the run and resets the next attempt to Floor 1 — Sublevel 0.

Difficulty Floors

Each floor has one fixed difficulty level, starting from Very Easy and becoming harder until Mastery Level. Each 12-question floor attempt uses 12 different Grade 10 Mathematics topics: 3 Algebra, 3 Geometry, 3 Probability, and 3 Statistics.

Quit Button

Use Quit Run to voluntarily stop your attempt. It does not count as a wrong answer.

Leaderboard

Students are ranked by floor reached, sublevel reached, then fastest time.
